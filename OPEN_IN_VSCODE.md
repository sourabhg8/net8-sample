# Open this project in VS Code

To see the **backend (Microservices API)** in VS Code, open one of these folders:

## Option 1 – Full repo (recommended)
Open the **solution root** (so the .sln and backend are both visible):

```
/Users/45111/Projects/net8-sample
```

- **File → Open Folder…** (or **Open…** on Mac) and choose the `net8-sample` folder above.
- You will see:
  - `Microservices.sln` (solution)
  - `src/Microservices/` (backend API – all changes are here)

## Option 2 – Backend only
Open only the backend project:

```
/Users/45111/Projects/net8-sample/src/Microservices
```

- Use **File → Open Folder…** and choose the `Microservices` folder inside `src`.
- You will see the API project only (no solution file in this folder).

---

If you open a **different** folder (e.g. another clone, or a folder named "backend" elsewhere), you will not see these changes. The backend code lives under **net8-sample/src/Microservices**.
